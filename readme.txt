-----------------------USER GUIDE TO USE THE LIBRARY MANAGEMENT SYSTEM-----------------------------------------------------------------

-----1)Install Mamp and start your Mamp server
-----2)Install the data.sql file for the database in phpmyadmin.	
-----3)Unzip the file pxm172730_cs6360.zip in the htdocs folder of mamp and type this in your url "localhost/pxm172730_cs6360/lib.html".
-----4)Use the UI which is displayed when you go to the link "localhost/pxm172730_cs63630/library/lib.html" to navigate through various modules like searching the books,add borrowers,return book and check fine/pay fines.
